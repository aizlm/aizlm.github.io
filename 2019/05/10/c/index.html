c程序的编译

得到一个c程序文件(pwnable.kr上的第五题passcode)

gcc格式: gcc [options][filename]

[options]-E 预处理阶段 不编译
gcc -E passcode.c -o passcode.i
less passcode.i

-S 将c程序编译为汇编文件 默认后缀.s
gcc -S passcode.c
less passcode.s

-c 将C源文件或者汇编语言文件编译成可重定向的二进制文件 默认后缀.o
gcc -c passcode.s -o passcode.o
less passcode.o 乱码
vim passcode.o 乱码 Esc:%!xxd转换为16进制形式 退出 Esc:%!xdd -r + Esc:q!

链接阶段:
gcc passcode.o 无任何options值 生成a.out文件
gcc passcode.o -o passcode 最终文件

调试:
gdb passcode 调试最终文件 a.out / passcode

32位环境下的安装包:
sudo apt-get install build-essential module-assistant
sudo apt-get install gcc-multilib g++-multilib
gcc -m32 passcode.c
