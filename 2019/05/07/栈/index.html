栈

python学习:https://blog.csdn.net/zzq900503/article/details/80538346

栈 from 清风阁

先认识几个汇编指令：

  • mov ax,1000 #AX=1000 传值
  • add ax,bx #AX=AX+BX 加
  • {push ax push bx push cx 压栈
  • pop ax pop bx pop cx} #AX=CX BX=BX CX=AX 出栈

    使用pop指令并不会删除当前栈顶的数据（只是复制，并SP=SP-2（十六进制2刚好是一个字节））
    但是在下一次push操作后会覆盖掉这个元素

    栈顶越界

    两种情况
  • 栈满时 使用push指令
  • 栈空时 使用pop指令

    EAX、ECX、EDX、EBX：为ax,bx,cx,dx的延伸，各为32位元
    EAX 是”累加器”(accumulator), 它是很多加法乘法指令的缺省寄存器。
    EBX 是”基地址”(base)寄存器, 在内存寻址时存放基地址。

在一次函数调用中,堆栈中将被依次压入:参数,返回地址,EBP.在函数返回时,弹出EBP,恢复堆栈到函数调用的地址,弹出返回地址到EIP以继续执行程序.

#在c程序中,参数的压栈顺序是反向的.
